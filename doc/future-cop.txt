INTRODUCTION                               *futurecop* *futurecop-introduction*

Vim plugin to run the gem future_cop on current file and display the results
in quickfix window

CONFIGURATION                                         *futurecop-configuration*

There are no configuration options at the moment.

Pull requests at http://www.github.com/futureworkz/vim-future-cop are welcome!

COMMANDS                                              *futurecop-commands*

:FutureCop                                            *:FutureCop*
Command to run FutureCop on current file and output it to quickfix
